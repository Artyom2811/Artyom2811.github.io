Шпаргалка:

TRIM всей колонки:

    UPDATE имя_таблицы
    SET имя_колонки = TRIM(имя_колонки)

Проверка на дубликаты:

    SELECT ‘проверяемая колонка’ , COUNT(*)
    FROM ‘проверяемая таблица’
    GROUP BY ‘проверяемая колонка’
    HAVING COUNT(*) > 1

CAST:

        Date to Text:
        to_char(имя_колонки, 'yyyy-MM-dd')

Заменить если null на Значение:

        COALESCE(имя_колонки, 'Нет значения')

Если нужно упорядочить по дате, если дата строкой(Нужно в сортировке перевести в дату):

        SELECT *
        FROM table
        ORDER BY DATE(date_coloumn) DESC

Статистика запуска скпиптов с индексом и без:

        SELECT
            relname,
            seq_scan,
            idx_scan
        FROM pg_stat_all_tables
        WHERE schemaname = 'имя_схемы'

Получение grant для на таблицу:

        GRANT SELECT, INSERT, UPDATE, DELETE ON имя_таблицы TO пользователь
        Пример:
        grant select, insert, update, delete on table1 to IVAN
