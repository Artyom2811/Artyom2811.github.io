H2

Что это:

Имеет свой личный веб интерфейс. Порт по умолчанию 81(localhost:81).

Установка через докер:

Получаем образ их репозитория:

docker pull oscarfonts/h2

Запускаем с портами(1521 для соединения, 81 дял веб интерфейса):

docker run -d -p 1521:1521 -p 81:81 -v /path/to/local/data_dir:/opt/h2-data --name=MyH2Instance oscarfonts/h2

Теперь можно настраивать через "localhost:81"
