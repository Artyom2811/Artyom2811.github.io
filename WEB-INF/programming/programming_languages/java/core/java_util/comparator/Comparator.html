Что это

Comparator - интерфейс для реализации сравнения объектов.

Контракт компаратора:

  • Отрицательный int - первый объект меньший.
  • Положительный int - первый объект больший.
  • Ноль - объекты равны.

Comparator vs Comparable

Если мы создаем класс реализуем Comparable.

Если класс создан не нами - мы можем использовать Comparator.
