Что это

Comparator - интерфейс для реализации сравнения объектов.

Только один метод compareTo(Object obj), предназначенный для реализации упорядочивания объектов класса. Делает проверку на совместимость и возвращает -1 если меньше 0 если равно и +1 если больше.

Контракт компаратора:

  • Отрицательный int - первый объект меньший.
  • Положительный int - первый объект больший.
  • Ноль - объекты равны.

Comparator vs Comparable

Если мы создаем класс реализуем Comparable.

Если класс создан не нами или мы хотим несколько вариантов сортировок - мы можем использовать Comparator.
