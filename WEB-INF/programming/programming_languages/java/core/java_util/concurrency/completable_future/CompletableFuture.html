Что это

CompletableFuture класс для асинхронного программирования. CompletableFuture это расширение Future.

Future был хорошим шагом на пути к асинхронному программированию, но ему не хватало некоторых важных и полезных функций.

Ограничения Future

  • Его нельзя завершить вручную.
  • Нельзя выполнять дальнейшие действия над результатом Future без блокирования. Также в Future нельзя повесить функцию-колбэк, чтобы она срабатывала автоматически, как только станет доступен результат.
  • Невозможно выполнить множество Future один за другим.
  • Невозможно объединить несколько Future.
  • Нет обработки исключений.

Методы

.allOf(...) - можно дождаться завершения сразу нескольких асинхронных задач, без ручной возни с join() по каждой.
