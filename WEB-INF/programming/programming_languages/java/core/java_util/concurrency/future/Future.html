Future

Что это

Future используется как ссылка на результат асинхронной задачи.

Методы:

  • boolean isDone() - проверить, закончилась ли задача
  • boolean isCancelled()
  • boolean cancel(boolean mayInterruptIfRunning) - попытаться отменить выполнение
  • V get() - блокирует поток до тех пор, пока результат не станет доступным.
  • V get(long timeout, TimeUnit unit)
