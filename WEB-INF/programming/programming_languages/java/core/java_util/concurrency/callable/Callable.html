Callable

Что это

Callable - очень похож на интерфейс Runnable, но может вернуть результат в виде объекта Object и способен бросать исключения.

Callable.call() - возвращает значения типа T или проверяемое исключение.

V call() throws Exception - метод для имплементации.

Значение возвращается во Future.

Future - Если вызвать метод get на не законченном Future, будет ожидание потока до выполнения(блокировка).
