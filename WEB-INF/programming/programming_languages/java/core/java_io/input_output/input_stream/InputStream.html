InputOutput

Что это

InputStream - представляет классы, для получения данных из источников.

  • Массив байтов.
  • String.
  • Файл.
  • Канал(pipe).
  • Последовательность потоков.
  • Другие источники(Интернет).

Реализации

  • ByteArrayInputStream
  • StringBufferInputStream - Deprecated. Use StringReader.
  • FileInputStream
  • PipedInputStream
  • SequenceInputStream
