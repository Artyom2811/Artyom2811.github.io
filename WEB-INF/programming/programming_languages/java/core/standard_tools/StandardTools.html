Преобразования

Восходящее

Восходящее - преобразование от потомка к предку.

Нисходящее

Нисходящее - преобразование от предка к потомку. Должно указываться явно с помощью указания нового типа в скобках. 

int x = 100;
byte y = (byte) x;

или

Animal dog = new Dog(); //восходящее преобразование. Будет потерян доступ ко всем методам, которые есть
        только у класса Dog.

if

if(условие) {
    тело(блок)
} else if  (условие) {
    тело(блок)
} else if (условие) {
    тело(блок)
} else {
    тело(блок)
}

Циклы(loops)

while

while(условие) {
    тело(блок)
}

do while

do {
    тело(блок)
} while (условие)

for

for (инициализация; условие; инкремент/декремент) {
    System.out.println(i);
}
    
for (int i = 0, j = 0; i < 10; i++) {
    System.out.println(i);
}

Бесконечные циклы - while(true), for(;;)

Switch

Если не стоит break выполниться несколько case.

Типы данных:byte, short, char, int, Byte, Short, Character, Integer, String, Enum.

Синтаксис

switch(выражение) {
    case значение1 : команда; break;
    case значение2 : команда; break;
    case значение3 : команда; break;
    default : оператор;
}

Text Blocks(since v.15)

Указание текста в том виде в котором он должен храниться(с переносами строки).

Синтаксис


String b ="""
1
2
3
""";

Вставка значений с строку

String a = "1";

"""
    Some parameter: %s
""".formatted(a);

Экранирование

Отмена новой строки(\)

"""1 \
   2
""";

Отмена новой строки(\s)

По умолчанию идет обрезка пробелов в каждой строке.

"""1      \s
   2
""";
