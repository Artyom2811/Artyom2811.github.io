Static

Может быть применён

  1. Внутренний класс - статический класс
  2. Блок инициализации
  3. Метод
  4. Поле

На static методы накладываются ограничения:

  1. Они могут вызывать только другие статические методы.
  2. Они должны осуществлять доступ только к статическим переменным.
  3. Они не могут ссылаться на члены типа this или super.
  4. Нельзя переопределить статический метод - переопределение зависит от наличия экземпляра класса, статический метод не ассоциируется ни к каким экземпляром класса.
