Подключение к Spring boot

Добавление зависимостей

implementation 'org.liquibase:liquibase-core:4.22.0'

Добавляем файл инициализации по пути - resources.db.changelog.db.changelog-master.xml

Или указать путь параметром - spring.liquibase.change-log=

Отключение через Spring properties
