DispatcherServlet - главный контроллер который принимает все сообщения и отправляет их выдирая из @RequestMapping.

Spring MVC зависимости(spring-webmvc) - содержит в себе зависимости:

  • spring-web
  • spring-core
  • spring-beans
  • spring-context
  • spring-aop
  • spring-expressions
  • spring-jcl
