Hotkeys

  • ALT + Enter - Отображение свойств выбранного элемента.
  • F2 - переименование.
  • Ctrl + N - дублировать окно.
  • Ctrl + Alt + N - создать новую папку.
  • Shift + Win + S - скриншот части экрана.

CMD

Основные команды:

    Каталог

  • dir - просмотр каталога в виде списка.
  • tree - вывод каталога в графическом представлении.
  • cd - смена текущего каталога.
  • md - создание каталога.
  • rd - удаление каталога.
  • chdir - отобразить текущую локацию.

    • Файл

  • type nul > name.txt - создание файла.
  • type - отобразить контент файла.
  • move - перемещение файла.
  • ren - переименование файла.
  • copy - копировать файл.
  • del - удалить файл.
  • fc - сравнить содержимое файлов.
  • find - поиск по строке в файле.
  • attrib - изменить разрешение доступа к файлу.

    • Общие

  • echo - вывод на экран.
  • time - отобразить время.
  • date - отобразить дату.
  • cls - очистить консоль.
  • help - вывод доступных команд.
  • exit - закрыть консоль.
  • tasklist - отобразить список процессов.
  • taskkill - завершить процесс.
  • set - отобразить все переменные среды.
  • set var=value - объявить переменную.
  • at - установить выполнение по расписанию.
  • ping - выполнить пингование по адресу.
  • netstat -ano - отобразить список открытых портов.
  • TAB - дополняет набираемую команду из списка доступных.

Environment variables

System environment variables

System environment variables - переменные имеют глобальный доступ для всех пользователей.

User environment variables

User environment variables - специфичны только для текущего пользователя, вошедшего в систему.
