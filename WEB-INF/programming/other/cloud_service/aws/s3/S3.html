Basic entities

    Bucket - папка в который будет храниться. Пример: photos.
    Object - объект, который будет храниться. Состоит из Key(name) и VersionID привер:folder/Bird.jpg.
    Link Address - путь к созданному файлу пример: https://s3.amazonaws.com/photos/folder/Bird.jpg.
