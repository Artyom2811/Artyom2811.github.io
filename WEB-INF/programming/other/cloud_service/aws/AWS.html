Что это

AWS - предоставляет услуги как по инфраструктурной модели (виртуальные серверы, ресурсы хранения). В данный момент есть более 100 сервисов.

Самые популярные сервисы Amazon

  • EC2(Elastic Compute Cloud) - Virtual Servers
  • Elastic Load Balancing - для распределения нагрузки между сервисами.
  • Elastic Beanstalk - для автоматизации deploy. DevOps Automation
  • CloudFront - отдельные от основных(Availability Zone) сервера, необходимые для кеширования, которое ускоряет получения информации.
  • CloudWatch - для мониторинга серверов.
  • VPC(Virtual Private Cloud) - Virtual Network
  • Amazon S3 - simple storage service. Для обмена информацией. Share service. Можно превратить в статический сайт.
  • Glacier - архивирование данных. Очень дешевый, но не быстрый.
  • EFS(elastic file system) - объединения нескольких серверов к одному жесткому диску.
  • S3 Bucket - Unlimited Web Space
  • S3 Web Hosting - Web Sites Hosting
  • CloudFormation - Infrastructure as Code. Используется для описания инструкций работы всех облачных сервисов.
  • OpsWorks - для автоматизации инфраструктуры. Загружает recipes для Chef(система управления конфигурациями).
  • RDS(Relational Database Service) - база данных SQL.
  • DynamoDB - база данных NoSQL.
  • Redshift - анализ больших данных.
  • EMR(Elastic MapReduce) - bigdata.
  • IAM(Identity and Access Management) - пользователи и доступы
  • Route 53 - Domain Registration, DNS. Отдельные сервера. Работает через порт 53.
  • SES(Simple Email Service) - сервер email.
  • SNS(Simple Notification Service) - посылаем email, sms и тд.
  • SQS(Simple Queue Service).

Расположение серверов

Region - географическое место расположение.

Availability Zone - изолированный дата центр.

Пример: region(us-west-1), availability zone(us-west-1a, us-west-1b, us-west-2c)
