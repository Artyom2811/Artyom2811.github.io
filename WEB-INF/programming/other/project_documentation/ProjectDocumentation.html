Project Documentation

Что это

Основные разделы

  • readme.txt - краткое описание проекта, словарь терминов, описание команды.
  • todo.txt
  • functional.drawio - mindmap для понимания и отслеживания необходимого функционала.
  • work_flow.txt - описание порядка выполнения заданий с нормами и отчетами.
  • error.txt - частые проблемы и способы решения.
  • meetings.txt - описание всех регулярных встреч.
  • responsibility.txt - расписать ежедневные обязанности для себя.
