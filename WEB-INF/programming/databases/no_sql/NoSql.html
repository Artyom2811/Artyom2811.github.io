Cassandra

Cassandra - имеет node которые обмениваются данными между собой.

CQL - язык запросов для cassandra.

allow filtering - команда для выполнения запроса с фильтрацией колонок в произвольном порядке.

toTimestamp() - метод для получения timestamp из строки. Пример: toTimestamp('2020-01-01')
