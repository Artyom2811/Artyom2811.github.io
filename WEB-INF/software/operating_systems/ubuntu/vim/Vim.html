VIM

VIM — свободный текстовый редактор, созданный на основе более старого vi.

Основные команды:

  1. :q - выход
  2. :i - включение режима редактирования.
  3. :w - сохранение
  4. :wq - сохранение и выход
  5. :E - Explore. Режим навигации.
  6. :set number - включить нумерацию строк.
  7. :set nonumber - выключить нумерацию строк.
