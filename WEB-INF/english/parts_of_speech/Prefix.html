Prefix

Dis

Dis - поменять на противоположное.

  1. disagree - не соглашаться
  2. dislike - неприязнь
  3. disconnect - отсоединиться
  4. dishonest - нечестный

Out

Out - сделать лучше, превосходить.

  1. outrun - обогнать
  2. outgrow - перерасти
  3. outwit - перехитрить
  4. outperfect - превзойти

Re

Re - повторить действие.

  1. rewrite - переписать
  2. remake - переделать
  3. reconsider - переосмыслить
  4. recreate - воссоздать

De

De - убрать или отменить действие.

  1. demotivate - демотивировать
  2. deactivate - деактивировать
  3. declassify - рассекретить
  4. detached - отдельный

Un

Un - отрицательный.

  1. unhappy - несчастный
  2. unable - неспособный
  3. unknown - неизвестный
  4. unclear - неясный

Under

Under - недовыполнять.

  1. underestimate - недооценивать
  2. underweight - недовес
  3. undersecretary - заместитель министра
  4. underpay - недоплачивать

Mis

Mis - неправильно.

  1. misbehave - плохо себя вести
  2. misspell - допускать орфографические ошибки
  3. mistake - ошибаться
  4. misunderstand - неправильно понимать

Ex

Ex - бывший/предыдущий.

  1. ex-wife - бывшая жена
  2. ex-president - бывший президент
  3. ex-champion - бывший чемпион
  4. ex-boyfriend - бывший парень
