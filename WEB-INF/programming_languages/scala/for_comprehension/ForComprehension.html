For-Comprehension

For-Comprehension - запись для выражения последовательных преобразований.

Имеет синтаксис - "for (enumerators) yield e". Можно использовать без "yield"

    enumerator может быть:
  1. Генератор - <-
  2. Фильтр - if

Также после первого генератора можно объявлять переменные через "=". Пример: userId: String = createdUser.id

Первый генератор устанавливает правило для For-Comprehension - если это List, то все остальные генераторы должны быть List и yield обернет ответ в List.

    В For-Comprehension можно работать с типами данных которые поддерживающий операции withFilter, map, and flatMap.
  1. Option
  2. List
  3. Future
  4. etc
