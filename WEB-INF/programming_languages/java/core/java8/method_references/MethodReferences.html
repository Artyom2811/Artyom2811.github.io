Method References(::)

Проработать:

  1. Ссылки на методы. Виды ссылок на методы. Ссылка на статический метод. Ссылка на метод экземпляра

Method References(ссылки на методы) - это сокращенный синтаксис выражения лямбда, который выполняет только один метод.

Основные виды:

  1. Ссылки на статические методы - имя_класса::имя_метода
  2. Ссылки на методы экземпляра:
    1. имя_экземпляра::имя_метода
    2. имя_класса::имя_метода
  3. Ссылки на конструкторы - имя_класса::new
  4. Ссылки на обобщенные (шаблонные) методы -
