Properties

Что это

Properties - стандартный класс для использования property в приложение.

Быстрый старт

Необходимо создать файл с конфигурацией пример названия(application.properties), пример файла:

email=test@gmail.com

Пример указания property файла:

Properties prop = new Properties();
try {
    prop.load(new FileReader("./src/main/resources/application.properties"));
} catch (IOException e) {
    e.printStackTrace();
}

Пример использования property:

//если точно указывать параметр, если не найдёт вернется null
prop.getProperty("email");

//указываем значение, которое будет использовано, если не будет найдено в файле настроек
prop.getProperty("email1", "didn't find");
