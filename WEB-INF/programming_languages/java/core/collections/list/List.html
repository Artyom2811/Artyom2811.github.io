Что это

List(список) - коллекция в которой элементы хранится по индексам. Также предоставляет доступ к элементам по значению. Допускает дубликаты.

Сложность алгоритмов ArrayList

  1. Получение по индексу - O(1)
  2. Поиск - O(n)
  3. Вставить - O(n)
  4. Удаление - O(n)

Сложность алгоритмов LinkedList

  1. Получение по индексу - O(1), получение по значению O(n)
  2. Поиск по индексу - O(1), поиск по значению O(n)
  3. Вставить - в начало или конец - O(1), в середину по индексу - O(1), по значению - O(n).
  4. Удаление - из начало или конец - O(1), в середину по индексу - O(1), по значению - O(n).

Основные реализации

  1. ArrayList
  2. LinkedList
  3. Vector
  4. CopyOnWriteArrayList - это потокобезопасный вариант of ArrayList. Разница в том, что все мутативные операции, такие как add, set, remove, clear,... создают новую копию массива.

ArrayList

При добавлении в середину списка создаётся новый массив в него помещается:
первая часть -> новый элемент -> вторая часть

LinkedList

Элементы хранятся в объекте Node. Которые имеют ссылки на предыдущий и следующий элементы.

Быстрый способ создания immutable collections(JAVA 9).

После создания никак нельзя изменить(add, remove, put). Но элементы в коллекции могут быть изменены. Нельзя добавлять null(NullPointerException).

  1. List.of("Luke", "Eva", "John");
  2. Map.of("name", "Luke", "age", "23", "city", "New York");
  3. Set.of("Luke", "Eva", "John");
