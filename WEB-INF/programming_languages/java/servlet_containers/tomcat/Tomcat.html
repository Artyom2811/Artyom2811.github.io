Tomcat 8

Что это:

Tomcat (в старых версиях - Catalina) - контейнер сервлетов с открытым исходным кодом. Реализует спецификацию сервлетов, спецификацию JavaServer Pages (JSP) и JavaServer Faces (JSF). Написан на языке Java.

Tomcat port: 8080

Для запуска Tomcat:

Необходимо запустить /bin/startup.sh

Для остановки Tomcat:

Необходимо запустить /bin/startup.sh

Для запуска war приложения, необходимо поместить его в /webapps. И tomcat распакует его и запустит.

Пользователи и роли настраиваются в файле /conf/tomcat-users.xml. Для доступа в определенные разделы на вебсайте Tomcat GUI, необходимо создать пользователей с определёнными ролями.

Вывод в консоль приложения находиться:

/logs/catalina.out
