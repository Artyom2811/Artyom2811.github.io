Работа с терминалом:

  1. clear - очистить историю терминала.
  2. uname -a - получить информацию о системе.
  3. top - получить информацию процессах.
      Горячие клавиши:
    • ? - help
    • L - поиск
    • z - разноцветный режим
    • x - подсвечивание столбца с сортировкой
    • <, > - выбор поля для сортировки
    • V - режим отображение "дерево"
    • k - kill by PID
    • W - сохранение настроек
    Команды навигации в Terminal
  1. ls -la - отобразить список файлов и папок в текущей директории
  2. cd - выполнить переход из текущей директории
  3. pwd(print working directory) - вывести текущую директорию.
    Поиск файла или папки:
  1. locale - поиск по имени. Но перед поиском необходимо выполнить команду "sudo updatedb" для обновления индексов.
  2. find /. -name "abc" -type d - поиск папки abc из корневой директории.
    Работа с файлами:
  1. tail -f -n 5 syslog - открыть последние 5 строк файла syslog с дальнейшим отслеживание.
  2. cp - копирование файла. Параметры: указать путь файл, указать путь куда копировать
    3. cp ./hosts-m3-dev-artyom ./hosts-m3-dev-artyom-backup
    Работа с VIM:
  1. :q - выход
  2. :w - сохранение
  3. :wq - сохранение и выход
  4. :E - Explore. Режим навигации.
    Работа с сетью:
  1. ping - пинг по ip. Пример:ping 8.8.8.8
  2. telnet - пинг по ip и порту. Пример:telnet opennet.ru 80
  3. netstat -ntlp - команда для отображение занятых портов.

Для просмотра файла через терминал можно использовать команду "сat". Пример:cat app.log
