Ubuntu - дистрибутив GNU/Linux, основанный на Debian GNU/Linux

Alt + F2 -> R - перезагрузка GNOME. Помогает если что-то с GNOME не так.

Основные разделы:

SSH

terminal

Настройки системы:

Выбор приложений по умолчанию - Setting -> Details -> Default Applications

Рекомендуемые программы для Ubuntu:

  • keepassx - программа хранения паролей
  • kazam - программа для записи экрана
  • pulseaudio и pavucontrol - для настройки звука (Output Devices -> Line Out -> 70%, Input Devices -> Rear Microphone -> 40%)
  • Pitivi - программа для редактирования видео
  • VLC - программа для просмотра видео

Работа с VIM:

  1. :q - выход
  2. :w - сохранение
  3. :wq - сохранение и выход
  4. :E - Explore. Режим навигации.
  5. :set number - включить нумерацию строк.
  6. :set nonumber - выключить нумерацию строк.

Работа с сетью:

  1. ping - пинг по ip. Пример:ping 8.8.8.8
  2. telnet - пинг по ip и порту. Пример:telnet opennet.ru 80
  3. netstat -ntlp - команда для отображения занятых портов.

Для просмотра файла через терминал можно использовать команду "сat". Пример:cat app.log
