Kafkacat

Имеет возможность регулярно посылать сообщения в kafka. Включение функции через терминал.
Пример: kcat -P -b m3n1:9092 -t batch -p 0 ./idea-recognition-sample_0_0.json

  • -P - режим продюсера
  • -b - режим брокера
  • -t - выбор топика
  • -p - выбор партишена
  • и указываем путь к файлу
