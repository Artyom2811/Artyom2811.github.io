Содержание

Коллекции- это специальные классы для хранения использования и удачение множества одинаковых объектов.

    Вопросы на собеседование:
  • Разница HashSet и LinkedHashSet? - LinkedHashSet хранит записи в порядке добавления, двухсвязный список.
  • Разница HashSet и TreeSet? - TreeSet на основе дерева. При добавлении сортирует. Для добавление необходимо передать Comparator или чтобы его элементы реализовывали Comparable.
  • Как получить значение из Hashset? Вызвать его Iterator или использовать метод toArray()
  • Как храняться эдементы в Hashset? В односвязном списке.
  • Что просходит при получении значение из Hashmap? Мы ищем по ключу баккет а дальше проверяем на equals() элементы пока не найдем.
  • Требование к ключу Map - Должен быть Immutable и преопределить equals(Object ob) и hashcode().
