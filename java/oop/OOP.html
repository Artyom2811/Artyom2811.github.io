Концепции объектно-ориентированного программирования JAVA

ООП - методология программирования, основанная на представлении программы в виде совокупности объектов, каждый из которых является экземпляром определённого класса, а классы образуют иерархию наследования. (способ програмирования, путем приравнивания частей кода с объектами из реального мира).

Принципы ООП:

Абстракция - означает построение классов исходя от их интерфейсов, и не смотря на их реализацию.

Наследование - процесс при котором один класс получает поля и методы другого.

Полиморфизм - один интерфейс — много реализаций. Например перегрузка или переопределение методов. Самый популярный пример с Animal и bark().

Инкапсуляция - сокрытие реализации. В Java регулируется при помощи модификаторов доступа. Все переменные должны быть private и доступ к ним должен осуществляться через set и get.
