Class

  1. Class - обычный класс для создания объектов.
  2. Abstract Class - абстрактный класс. Для описания того из чего нелься создать объект(например мебель).
    3. Внутрении:
    1. Nested Inner classes (вложенные внутренние классы)
      2. Создание объекта: Outer.Inner inner = new Outer().new Inner();
    2. Static Nested classes or Member of outer class (статические вложенные классы)
    3. Method Local Inner classes (внутренние классы в локальном методе)
    4. Anonymous Inner classes (анонимные классы)
  4. Interface - специальный класс который, контракт как докозательство того что класс имплементирующий его имеет реализации его методов. Имеет default method. Поля только static final.
  5. enum (Перечисления)- представляют набор логически связанных констант. Могу иметь: поля, конструкторы, методы, и инплементировать интерфейсы. Инициализация:Day current = Day.MONDAY;.

В классе могут быть:

  1. Поля
  2. Методы
  3. Конструкторы
  4. Объекты
  5. Блок инициализации
  6. Статический Блок инициализации - используется для инициализации нестатических переменных. В них можно выполнять методы.
  7. Внутрении классы - отличается словом static. Используется для инициализации статических переменных
