Generics

Generics - это подход к реализации класса, корорый позволяет использовать его с различными типами данных без изменения его описания. Необходим чтобы во время компиляции мы проверяли места где мы ждем определенные типы на соответствие этим типа. А также будет автокаст к указаному типу.

  • При сравнении Set<String> и Set<Integer> будет true. Тк. type erasure при компиляции заменить все на Object.

PECS - при использовании WILDCARD

  • Использовать коллекцию для получения, используем extend.
  • Использовать коллекцию для вложения, используем super.
  • Если то и другое то ничего не используем.
