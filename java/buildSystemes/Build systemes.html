Содержание

Apache MAVEN - фреймворк для автоматизации сборки проектов на основе описания их структуры в файлах на языке POM, являющемся подмножеством XML.

  • clean - очищяет все сгенерированые файлы.
  • validate - проверяет правильность конфигурации.
  • compale - компилирует исходные коды.
  • test - выполняет тесты.
  • package - создает инсталяционные пакеты.
  • verify - проверяет пакеты на целлостность.
  • install - устанавливает пакеты в локальный репозиторий.
  • site - генерирует документацию и javadoc.
  • deploy - выкладывает созданые пакеты во внешний репозиторий.
