Содержание

Сериализация — это процесс сохранения состояния объекта в последовательность байт. В JAVA для сериализации объекта мы должны пометить его интерфейсом Serializable. Далее мы можем использовать ObjectOutputStream.writeObject() для его сериализации или десериализации ObjectInputStream.readObject().
Если мы не хотим сериализовывать какое-то поле мы используем ключевое слово transient.
После имплементации Serializable мы можем инициализировать переменную long serialVersionUID. Значение этого поля говорит нам о версии нашего класса. Она задается автоматически(не рекомендуется) или вручную. Пример мы сделали сериализацию, изменили класс и делаем десереализацию в уже измененый класс - нам вернется исключение InvalidClassException. Тоесть при изменения класса не забываем перегенерировать с помощью ide версию класса.
