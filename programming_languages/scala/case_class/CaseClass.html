Case Class

Case Class(класс образец) - служат для создания неизменяемых объектов, которые используются за POJO.

Объект создаётся без ключевого слова new, тк по умолчанию имеют объект компаньон с методом apply.

Пример объявления case class и создания объекта:

case class Book(isbn: String)
val f = Book("1")

== - метод сравнения объектов по структуре.

copy - метод для создания копии объекта. Пример:

case class Employee(name: String, office: String, role: String)
val fred = Employee("Fred", "Anchorage", "Salesman")
val joe = fred.copy(name="Joe")
