JAVA - объектно ориентированный язык программирования. Java программы выполняются в пространстве виртуальной машины Java. Java программы компилируются в байт-код(javac), который при выполнении интерпретируются виртуальной машиной для конкретной платформы (динамическая компиляция (JIT - Just in time)). Компиляция — трансляция программы, составленной на исходном языке высокого уровня, в эквивалентную программу на низкоуровневом языке, близком машинному коду (байт-код, ассемблер и т.д.) Интерпретация — построчный анализ, обработка и выполнение исходного кода программы или запроса. По форме байт-код похож на машинный код, но предназначен для исполнения не реальным процессором, а виртуальной машиной.
