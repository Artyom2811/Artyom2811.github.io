Как она работает?

Отличие от конкурентов?

Быстрый старт:

Добавить зависимость:

<dependency>
    <groupId>log4j</groupId>
    <artifactId>log4j</artifactId>
    <version>set_last_version</version>
</dependency>

Добавить файл конфигурации /src/main/resources/log4j.properties

Задать начальные конфигурации.

log4j.rootLogger=DEBUG, stdout
log4j.appender.stdout=org.apache.log4j.ConsoleAppender
log4j.appender.stdout.layout=org.apache.log4j.SimpleLayout
log4j.logger.com.me.proj2=INFO

Добавить в логируемый класс объект для логирования.

import org.apache.log4j.Logger;
public class App {
    private static final Logger logger = Logger.getLogger(App.class);
}

Можно использовать:

logger.debug("Logger added");
