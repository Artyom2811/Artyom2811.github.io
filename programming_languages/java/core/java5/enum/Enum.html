Enum

    Особенности:
  1. Может иметь конструктор, методы, в том числе скрытые и абстрактные.
  2. Может реализовывать интерфейс.
  3. Можно использовать в операторе switch.
    Методы:
  1. values() - возвращает массив, содержащий список констант перечисления.
enum Cat {
    Leopard, Puma, Lion, Tiger, Manul
}
