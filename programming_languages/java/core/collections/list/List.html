List

Проработать:

  1. List в Java

Что это:

List(список) - коллекция в которой элементы хранится по индексам.

Особенности:

  1. Использует индексацию.
  2. Возможны дубликаты.

Основные реализации:

  1. ArrayList
  2. LinkedList
  3. Vector
  4. CopyOnWriteArrayList - это потокобезопасный вариант of ArrayList. Разница в том, что все мутативные операции, такие как add, set, remove, clear,... создают новую копию массива.

ArrayList

При добавлении в середину списка создаётся новый массив в него помещается:
первая часть -> новый элемент -> вторая часть

LinkedList

Элементы хранятся в объекте Node. Которые имеют ссылки на предыдущий и следующий элементы.
