Immutable Class

Immutable Class - класс который невозможно изменить после инициализации.

  1. Объявите класс как final, чтобы он не мог быть расширен.
  2. Сделайте все поля приватными, чтобы прямой доступ не допускался.
  3. Не писать Сеттеры.
  4. Сделайте все mutable поля final, чтобы их значение могло быть присвоено только один раз
  5. Инициализируйте все поля через конструктор, выполняющий глубокое копирование.
  6. Выполните клонирование объектов в геттерах.
