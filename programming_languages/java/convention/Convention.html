Convention

Конвенция - соглашение между програмистами, для общей выгоды.

    Именнование:
  1. Пакеты - lowercase. Компании используют свое инвертированное имя Интернет-домена, чтобы начать их имена пакета — например, com.example.mypackage
  2. Классы и интерфейсы - существительное PascalCase.
  3. Методы - глагол camelCase
  4. Переменные - camelCase. Не начинаться с "_" или "$"
  5. Константы - SCREAMING_SNAKE_CASE
